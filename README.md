# Gear-Box-Test-Rig-Anomaly-Detection
In this Gear Box Test Rig Anomaly Detection, the gear box may have different types of failures for example : Gear faul, Bearing fault, Shaft misalignment / imbalanace, Lubrication failures , Housing resonance issues and so on. 
The gearbox is mainly used for load, speed , temperature tests.
# Libraries-Required
scikit learn
pandas 
numpy
# Anomaly Detection
For the anomaly detection we use different readings to detect the most unusual reading that we get from the data available
